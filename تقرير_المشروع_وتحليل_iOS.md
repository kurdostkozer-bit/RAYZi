# تقرير مشروع Rayzi Ultimate - تحليل ومتطلبات نظام iOS

## 📋 نظرة عامة على المشروع

**Rayzi Ultimate** هو منصة ألعاب اجتماعية وألعاب حظ عبر الإنترنت متكاملة، تتضمن:

### الألعاب الحالية
- TeenPatti (لعبة ورق هندية)
- Roulette Casino (لعبة الروليت)
- FerryWheel (عجلة الحظ)
- Agency (نظام الوكالة)
- Host (نظام المضيفين)

### المنصات الحالية
- ✅ تطبيق Android (متوفر)
- ✅ تطبيق ويب (React)
- ❌ تطبيق iOS (غير متوفر - يحتاج تطوير)

---

## 🏗️ البنية التقنية الحالية

### Backend
- **اللغة**: Node.js
- **الإطار**: Express.js
- **الاتصال الحي**: Socket.io
- **قاعدة البيانات**: MongoDB
- **إدارة العمليات**: PM2

### Frontend
- **الإطار**: React
- **الإدارة الحالة**: React State Management
- **البناء**: Webpack

### الخادم
- **Web Server**: Nginx
- **SSL**: Let's Encrypt
- **نظام التشغيل**: Ubuntu 22.04
