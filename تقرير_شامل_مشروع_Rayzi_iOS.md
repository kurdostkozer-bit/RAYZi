## نظرة عامة على المشروع

**Rayzi Ultimate** هو منصة ألعاب اجتماعية وألعاب حظ عبر الإنترنت متكاملة.

### الألعاب المتاحة
- TeenPatti (لعبة ورق هندية)
- Roulette Casino (لعبة الروليت)
- FerryWheel (عجلة الحظ)
- Agency (نظام الوكالة)
- Host (نظام المضيفين)

### المنصات الحالية
- ✅ تطبيق Android (متوفر)
- ✅ تطبيق ويب (React)
- ✅ Backend (Node.js + MongoDB)
- ✅ لوحة تحكم (Admin Panel)
- ❌ تطبيق iOS (تم تطوير الكود - يحتاج بناء)

## ما تم إنجازه

### ✅ الكود المكتوب بالكامل (90% جاهز)

#### الملفات والهيكل
```
Source Code/iOS App/
├── src/
│   ├── assets/           # الصور والخطوط
│   ├── components/       # 7 مكونات مشتركة
│   ├── navigation/       # نظام التنقل الكامل
│   ├── screens/          # 13 شاشة
│   ├── services/         # 5 خدمات
│   ├── store/           # Redux Store (5 slices)
│   └── utils/           # الأدوات المساعدة
├── App.js              # نقطة الدخول
├── package.json        # المكتبات
└── التوثيق الشامل
```

#### المكونات المشتركة (7)
1. **Button** - أزرار متعددة الأنماط والألوان
2. **Input** - حقول إدخال مع تحقق
3. **Card** - بطاقات عرض مع تدرجات لونية
4. **Modal** - نوافذ منبثقة
5. **Loading** - مؤشرات تحميل
6. **GameCard** - بطاقات الألعاب
7. **CoinDisplay** - عرض العملات

#### الشاشات (13 شاشة)

##### المصادقة (4 شاشات)
1. **SplashScreen** - شاشة البداية
2. **LoginScreen** - تسجيل الدخول
3. **RegisterScreen** - إنشاء حساب
4. **ForgotPasswordScreen** - استعادة كلمة المرور

##### الرئيسية (1 شاشة)
5. **HomeScreen** - الشاشة الرئيسية مع عرض العملات والألعاب

##### الألعاب (6 شاشات)
6. **GamesListScreen** - قائمة الألعاب
7. **TeenPattiScreen** - لعبة Teen Patti
8. **RouletteScreen** - لعبة Roulette
9. **FerryWheelScreen** - عجلة الحظ
10. **TableSelectionScreen** - اختيار الطاولة
11. **GameRoomScreen** - غرفة اللعب

##### الأخرى (2 شاشة)
12. **WalletScreen** - شاشة المحفظة
13. **ProfileScreen** - الملف الشخصي

#### الخدمات (5 خدمات)
1. **api.js** - إعداد Axios مع interceptors
2. **authService.js** - خدمات المصادقة
3. **userService.js** - خدمات المستخدم
4. **walletService.js** - خدمات المحفظة
5. **socketService.js** - اتصال Socket.io

#### Redux Store (5 slices)
1. **authSlice** - حالة المصادقة
2. **userSlice** - حالة المستخدم
3. **gameSlice** - حالة الألعاب
4. **walletSlice** - حالة المحفظة
5. **socketSlice** - حالة Socket

#### نظام التنقل
- **Auth Stack** - مصادقة المستخدم
- **Main Tabs** - التنقل الرئيسي
- **Games Stack** - تنقل الألعاب
- **Stack Navigator** - التنقل المتداخل

## 🚀 خطوات البناء (تحتاج Mac)

### الخطوة 1: الحصول على Mac

#### الخيار A: Mac موجود
- استخدم Mac الخاص بك مباشرة

#### الخيار B: Mac Cloud (الأرخص)
- **MacInCloud**: $0.30/ساعة
- **MacStadium**: $0.50/ساعة
- **التكلفة**: 4-6 ساعات = $2-6 فقط

### الخطوة 2: إنشاء مشروع React Native فعلي

```bash
# على Mac
cd "Source Code/iOS App"
npx react-native@latest init RayziIOS
```

### الخطوة 3: نسخ الكود المكتوب

```bash
cd RayziIOS
cp -r ../src .
cp ../App.js .
cp ../index.js .
cp ../package.json .
cp ../app.json .
cp ../babel.config.js .
cp ../metro.config.js .
cp ../.gitignore .
```

### الخطوة 4: تثبيت المكتبات

```bash
npm install
cd ios
pod install
cd ..
```

### الخطوة 5: إعداد ملف البيئة

```bash
nano .env
```

```env
API_BASE_URL=https://your-admin-domain.com
SOCKET_URL=https://your-teenpatti-domain.com
ROULETTE_URL=https://your-roulette-domain.com
FERRYWHEEL_URL=https://your-ferrywheel-domain.com
SHARED_SECRET_KEY=your_secret_key
JWT_SECRET=your_jwt_secret
```

### الخطوة 6: التشغيل والاختبار

```bash
# على محاكي iOS
npx react-native run-ios

# أو عبر Xcode
open ios/RayziIOS.xcworkspace
```

### الخطوة 7: البناء للإطلاق

```bash
cd ios
xcodebuild -workspace RayziIOS.xcworkspace \
  -scheme RayziIOS \
  -configuration Release \
  -archivePath build/RayziIOS.xcarchive \
  archive
```

### الخطوة 8: الرفع إلى App Store

1. افتح Xcode Organizer
2. اختر الأرشيف
3. اضغط "Distribute App"
4. اتبع التعليمات
