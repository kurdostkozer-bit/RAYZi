## 🚀 خطوات البناء (تحتاج Mac)

### الخطوة 1: الحصول على Mac

#### الخيار A: شراء/استخدام Mac موجود
- إذا كان لديك Mac، استخدمه مباشرة

#### الخيار B: استئجار Mac Cloud (الأرخص)
- **MacStadium**: $0.50/ساعة
- **MacInCloud**: $0.30/ساعة
- **التكلفة التقديرية**: 4-6 ساعات = $2-6 فقط

---

### الخطوة 2: إنشاء مشروع React Native فعلي

#### على Mac:
```bash
# 1. الانتقال إلى مجلد المشروع
cd "Source Code/iOS App"

# 2. إنشاء مشروع React Native جديد
npx react-native@latest init RayziIOS

# 3. هذا سيكون ي:
# - إنشاء مجلد RayziIOS جديد
# - إنشاء ios/ و android/ folders
# - تثبيت المكتبات الأساسية
```

---

### الخطوة 3: نسخ الكود المكتوب

#### على Mac:
```bash
# 1. الانتقال إلى المشروع الجديد
cd RayziIOS

# 2. نسخ الكود من iOS App/src إلى RayziIOS/src
cp -r ../src .

# 3. نسخ الملفات الأساسية
cp ../App.js .
cp ../index.js .
cp ../package.json .
cp ../app.json .
cp ../babel.config.js .
cp ../metro.config.js .
cp ../.gitignore .
```

---

### الخطوة 4: تثبيت المكتبات

#### على Mac:
```bash
cd RayziIOS

# تثبيت المكتبات
npm install

# إعداد iOS
cd ios
pod install
cd ..
```

---

### الخطوة 5: إعداد ملف البيئة

#### إنشاء ملف `.env`:
```bash
cd RayziIOS
nano .env
```

#### محتوى `.env`:
```env
API_BASE_URL=https://your-admin-domain.com
SOCKET_URL=https://your-teenpatti-domain.com
ROULETTE_URL=https://your-roulette-domain.com
FERRYWHEEL_URL=https://your-ferrywheel-domain.com
SHARED_SECRET_KEY=your_secret_key
JWT_SECRET=your_jwt_secret
```

---

### الخطوة 6: التشغيل والاختبار

#### على محاكي iOS:
```bash
cd RayziIOS
npx react-native run-ios
```

#### على جهاز iOS حقيقي:
```bash
# 1. افتح Xcode
open ios/RayziIOS.xcworkspace

# 2. اختر جهازك من القائمة
# 3. اضغط زر التشغيل (▶)
```

---

### الخطوة 7: البناء للإطلاق

#### بناء Release:
```bash
cd RayziIOS/ios

# بناء الأرشيف
xcodebuild -workspace RayziIOS.xcworkspace \
  -scheme RayziIOS \
  -configuration Release \
  -archivePath build/RayziIOS.xcarchive \
  archive

# تصدير IPA
xcodebuild -exportArchive \
  -archivePath build/RayziIOS.xcarchive \
  -exportPath build/RayziIOS \
  -exportOptionsPlist ExportOptions.plist
```

---

### الخطوة 8: الرفع إلى App Store

#### عبر Xcode:
1. افتح Xcode Organizer
2. اختر الأرشيف الذي بنيته
3. اضغط "Distribute App"
4. اختر "App Store Connect"
5. اتبع التعليمات
