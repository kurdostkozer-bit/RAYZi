## 🚀 الخطوات التالية

### للبناء على جهاز Mac:

#### 1. تثبيت المتطلبات
```bash
# تثبيت CocoaPods
sudo gem install cocoapods

# تثبيت المكتبات
cd RayziIOS
npm install
```

#### 2. إعداد iOS
```bash
cd ios
pod install
cd ..
```

#### 3. التشغيل على المحاكي
```bash
npx react-native run-ios
```

#### 4. البناء للإطلاق
```bash
# افتح Xcode
open ios/RayziIOS.xcworkspace

# أو من سطر الأوامر
cd ios
xcodebuild -workspace RayziIOS.xcworkspace \
  -scheme RayziIOS \
  -configuration Release \
  -archivePath build/RayziIOS.xcarchive \
  archive
```

### 🔧 الإعدادات المطلوبة:

#### 1. ملف .env
```env
API_BASE_URL=https://your-admin-domain.com
SOCKET_URL=https://your-teenpatti-domain.com
ROULETTE_URL=https://your-roulette-domain.com
FERRYWHEEL_URL=https://your-ferrywheel-domain.com
SHARED_SECRET_KEY=your_secret_key
JWT_SECRET=your_jwt_secret
```

#### 2. إعدادات Xcode
- Bundle Identifier: `com.rayzi.ultimate`
- Team: Apple Developer Team
- Signing: Automatic
